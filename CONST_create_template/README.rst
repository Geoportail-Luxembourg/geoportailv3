geoportailv3 project
===================

Read the `Documentation <https://camptocamp.github.io/c2cgeoportal/2.5/>`_

Checkout
--------

.. code::

   git clone git@github.com:camptocamp/geoportailv3.git

   cd geoportailv3

Build
-----

.. code::

  ./build

Run
---

.. code::

   docker-compose up -d

.. Feel free to add project-specific things.
